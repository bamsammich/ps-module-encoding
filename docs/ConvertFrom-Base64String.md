# ConvertFrom-Base64String

## SYNOPSIS
ConvertFrom-Base64String decodes a base64-encoded string.

## SYNTAX

```
ConvertFrom-Base64String [-Base64String] <String> [<CommonParameters>]
```

## DESCRIPTION
ConvertFrom-Base64String decodes a base64-encoded string.

## EXAMPLES

### EXAMPLE 1
```
ConvertFrom-Base64String -Base64String cwB0AHIAaQBuAGcA
```
