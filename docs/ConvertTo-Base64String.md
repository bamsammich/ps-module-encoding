# ConvertTo-Base64String

## SYNOPSIS
ConvertTo-Base64String encodes a string to its base64 value.

## SYNTAX

```
ConvertTo-Base64String [-String] <String> [<CommonParameters>]
```

## DESCRIPTION
ConvertTo-Base64String encodes a string to its base64 value.

## EXAMPLES

### EXAMPLE 1
```
ConvertTo-Base64String -Base64String random
```
